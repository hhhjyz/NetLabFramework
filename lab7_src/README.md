## 实验环境

- 操作系统：macOS 26.1
- 编译器：
```
Apple clang version 17.0.0 (clang-1700.3.19.1)
Target: arm64-apple-darwin25.1.0
Thread model: posix
InstalledDir: /Library/Developer/CommandLineTools/usr/bin
```

## 4. 编译与运行

### 4.1 编译

在`lab7_src/`目录下，使用以下命令编译客户端和服务端：

```bash
make
```

### 4.2 运行

在多个终端中分别启动服务端和客户端：
```bash
# 启动服务端
./server

# 启动客户端
./client
```
